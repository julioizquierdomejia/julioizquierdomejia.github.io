Here are some sites that use Docpress.

Project sites

Docpress lends itself well to building clean, accessible project documentation. These open-source projects are using Docpress to generate their sites.

  • bumped - Make releasing software easy
  • dom101 - DOM helpers for JavaScript
  • ember-watson - Doctor for Ember Apps
  • hapi-decorators - Route decorators for Hapi.js
  • json2csv - Convert JSON to CSV
  • Onmount - Run something when a DOM element appears or exits
  • osom - Object schema modeling
  • tydel - Typed Models and Collections for JavaScript data structure
  • BotUI - A JavaScript framework to build conversational UIs

Books and manuals

Docpress isn’t just restricted to project documentation. Here are some online book-like websites generated using Docpress.

  • JS Mythbusters - A JavaScript optimization handbook
  • Mapgeo Help Manual - Users manual for the Mapgeo service
  • rscss - Reasonable System for CSS Stylesheet Structure
  • Basic Electronics - IIIT-Bangalore is using Docpress to make Course Material available to students.
