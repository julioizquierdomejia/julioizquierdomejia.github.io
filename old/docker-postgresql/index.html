Docker & PostgreSQL

01. Descargar la imagen de Docker

Todas las imagenes estan en el docker hub, y para bajar la imagen a nuestro computador solo debemos de escribir la siguiente sentencia en nuestra terminal 

% docker pull postgress

Es muy importante tambien indicar con que version de postgress vamos a tranajar, y debemos de indicar la version para poder descargar la imagen correcta, por ejemplo si vamos a descargar la version 9.6 la sentencia sería la siguiente 

% docker pull postgress:9.6

Listar imágenes descargadas

Una vez que ya este descargada, podemos ver las imagenes descargadas con el siguiente comando 

% docker images

en mi caso tengo muchas imagenes asi que utilizamos el siguiente comando para poder ver solo la imagen que queremos 

% docker images | grep postgres
